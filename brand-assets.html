Fonts

Main Font Family

To guarantee clear legibility and hierarchy across all languages, Manitou use one typeface.

Noto Sans

Noto Sans is the default font for all Manitou apps.

Check out Noto and its documentation on the website that has been developed by Google to promote this font.

Product Name Font

The Manitou Product Font is only used for product names. Use the .manitou-product-font in order to display text with it. Only UPPER CASE. Only for Manitou Equipment names.
